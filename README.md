# Java Programs Collection

This repository contains multiple Java programs developed as part of my internship tasks.

## Programs Included

1. ATM System – Simulates basic banking operations like deposit, withdraw, and balance check.
2. Calculator – Performs basic arithmetic operations.
3. Guess Number – A simple number guessing game.
4. Library Management System – Manages book records with add, search, and delete operations.
5. Login System – Validates user login credentials.
6. Student Grade System – Calculates and displays student grades.

## How to Run

1. Compile the program:
   javac FileName.java

2. Run the program:
   java FileName
